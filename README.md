## 👨🏻‍💻 Sobre Mim

Sou estudante do **4º semestre de Análise e Desenvolvimento de Sistemas (ADS)**, com formatura prevista para meados de 2027. Minha relação com a tecnologia nasceu da curiosidade de entender como as aplicações funcionam "por baixo dos panos", o que me ajudou a desenvolver uma base forte em lógica de programação e resolução de problemas.

Atualmente, estou direcionando meus estudos para o desenvolvimento **Backend**, mergulhando no ecossistema **Java**. Meu objetivo diário é absorver os fundamentos da Orientação a Objetos, modelagem de dados e boas práticas, preparando o terreno para construir APIs e sistemas escaláveis.

*   🎯 **Foco atual:** Domínio da linguagem Java, POO e modelagem de Bancos de Dados Relacionais (SQL/PostgreSQL).
*   🌱 **Próximos passos:** Aprofundamento no ecossistema Spring (Spring Boot) para desenvolvimento de APIs RESTful.
*   💻 **Minha Base:** Experiência prática com C e Python para lógica e automação, além de versionamento contínuo com Git.
